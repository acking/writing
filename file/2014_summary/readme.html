2014工作总结-Acking

支持平台：Mac、Pc、Mobile、Pad

支持浏览器：Firefox、Chrome、Safari、Internet Explorer9+

使用说明：

  • 翻页--->>支持方向键，上、下、左、右翻页（注：部分页面可上下翻页，见页面提示）
  • 放大--->>按住alt键+点击鼠标左键（移动端不支持）
  • 暂停--->>按b暂停当前页面（移动端不支持）
  • 预览所有幻灯片--->>按ESC键，方向键选择，回车键可进入选择的幻灯片（移动端不支持）
  • 全屏播放--->>按F11键全屏预览（移动端不支持）
